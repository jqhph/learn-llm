三、结构化输出与数据验证（Pydantic + Instructor）

工具一句话定位在 RAG 中的作用
PydanticPython 数据验证库（月下载 50M+）类型安全的配置管理、API 数据模型、LLM 输出 Schema 定义；v2 用 Rust 内核，验证快 5-50×
InstructorPydantic + LLM = 结构化提取传 Pydantic 模型给 LLM → 返回类型化 JSON；自动重试+验证+流式解析；支持 OpenAI/通义/Anthropic/Cohere
from pydantic import BaseModel
from instructor import from_openai

class CompanyInfo(BaseModel):
  company_name: str
  founded_year: int
  revenue: float | None = None

client = from_openai(OpenAI())
info = client.chat.completions.create(
  model="gpt-4o-mini",
  response_model=CompanyInfo,  # 声明式结构化提取
  messages=[{"role":"user","content":"..."}],
)
print(info.company_name)  # 类型安全，str
print(info.founded_year)  # 自动转为 int

九、Qdrant 快速启动

# 安装
pip install qdrant-client
docker run -d --name qdrant -p 6333:6333 qdrant/qdrant

# 创建集合 & 写入
from qdrant_client import QdrantClient
from qdrant_client.http.models import Distance, VectorParams, PointStruct

client = QdrantClient(host="localhost", port=6333)
client.create_collection("my_collection",
    vectors_config=VectorParams(size=768, distance=Distance.COSINE))
client.upsert("my_collection", points=[
    PointStruct(id=hash("doc1")%(2**63), vector=[0.1]*768,
                payload={"text": "文档内容", "category": "政策"}),
])

# 检索
results = client.search("my_collection", query_vector=[0.1]*768, limit=5)
