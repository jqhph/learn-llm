📐 BM25 关键词检索

核心参数

参数默认值作用调参方向
k11.2–2.0词频饱和度：越大则高频词贡献越大短文档 ↑，长文档 ↓
b0.75长度归一化：0=不惩罚长文档，1=完全惩罚文档长度差异大 ↑

安装

pip install rank-bm25 jieba

基础用法

from rank_bm25 import BM25Okapi
import jieba

tokenized = [list(jieba.cut(d)) for d in texts]
bm25 = BM25Okapi(tokenized, k1=1.5, b=0.75)
scores = bm25.get_scores(
    list(jieba.cut(query)))

🔀 RRF 分数融合

公式

分数(d) = Σ 1 / (k + rank_r(d))

k = 60（默认）
rank_r(d) 从 1 开始

Python 实现

def rrf_score(ranks, k=60):
    return sum(
        1.0 / (k + r) for r in ranks
    )

特性

  • 只看排名，不看分数
  • 天然支持异构排序器融合
  • k=30–100 结果稳定
  • 可与 Cross-Encoder 串联

Multi-Query 融合

def multi_query_retrieve(q, retrieve_fn):
    variants = llm_expand(q)  # → [q, v1, v2, v3]
    scores = {}
    for vq in variants:
        results = retrieve_fn(vq)
        for j, doc in enumerate(results):
            sid = doc["id"]
            scores.setdefault(sid, 0)
            scores[sid] += 1.0 / (60 + j + 1)
    return sorted(scores.items(),
                  key=lambda x: -x[1])
